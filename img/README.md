# Image Drop Guide — Jerome Baker Comes to New York

Drop images in this folder using the **exact filenames** below. Each scene gets up to 3 images (a/b/c). The page automatically picks them up — no code changes needed. Missing images show a styled placeholder card with the expected filename, so it's always obvious what's still needed.

## Recommended specs

- **Format:** `.jpg` (preferred) or `.png`
- **Hero image:** ~1600×900 (16:9)
- **Scene images:** ~1200×900 (4:3) or larger
- **File size:** keep under ~500KB each — compress with squoosh.app or similar

- `hero.jpg` — top of page, Jerome in NY establishing shot
